This is an experiment to see how many distinct sounds can be generated that the average human can hear.

# Dependencies:
* numpy
* pyaudio
  * Linux - portaudio19-dev, pythonX-dev

# Running:
* python main.py
* If using pipenv:
  * pipenv shell
  * python main.py
